# BOEM Corporate Site (Next.js)

Production-ready multilingual corporate website foundation for an international certification and compliance company.

## Stack
- Next.js App Router + TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form + Zod

## Locales
- `en` (default)
- `ru`
- `tr`
- `zh`

## Run
```bash
npm install
npm run dev
```

## Planned extensions
- Resend integration for forms
- CRM sync
